# Project Description

This project models a secure online exam proctoring system designed to maintain academic integrity in remote environments.

## Problem
Online exams lack reliable monitoring, making cheating difficult to detect.

## Solution
A system that:
- Verifies student identity
- Monitors behavior using webcam, audio, and screen
- Detects suspicious activity
- Generates detailed reports

## Technologies & Methods
- Model-Based Systems Engineering (MBSE)
- GENESYS tool
- UML, EFFBD, IDEF0, Activity, Sequence diagrams

## Outcome
A complete system-level design that demonstrates real-time monitoring logic and secure exam process modeling.
